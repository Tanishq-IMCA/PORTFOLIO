## About This Project

This is a personal portfolio website designed to showcase my skills as a front-end developer. The focus is on creating a clean, modern, and highly performant user experience with a heavy emphasis on animation.

The aesthetic is inspired by the idea of a "complete animator's toolbox," with a focus on smooth, mathematically precise animations. The goal is to create a website that is not only visually appealing but also a joy to interact with.

---

## Setup Instructions

### Prerequisites
-   Node.js 16+
-   npm or yarn

### 1. Install
```bash
npm install
```

### 2. Run the app
```bash
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).
